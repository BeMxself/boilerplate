# boilerplate
Using MVVM architecture(Rxswfit + Moya) to implement Github client demo.

## Features
*  Rxswift
*  Moya (networking layer)
*  MVVM architecture implement
*  TableView Pagination 
*  RxDataSources
*  ObjectMapper (Object mapping)
